# Roadmap

## Phase 0: Foundation

- Initialize Godot 4 project settings.
- Create project folders for scenes, scripts, data, docs, and assets.
- Add JSON schemas by convention through well-structured sample content.
- Add autoloads for content database, game state, and passport progression.

## Phase 1: MVP Prototype - Ha Giang Loop

- Implement player movement and camera follow.
- Load location, NPC, quest, and lesson content from JSON.
- Build interactable NPC placeholders.
- Show dialogue lessons in a reusable UI panel.
- Award English Passport stamps for lesson and quest completion.
- Validate the initial content set: 3 NPCs, 5 quests, 10 lessons.

## Phase 2: Playable Learning Loop

- Add quest acceptance and completion states.
- Add lesson choice prompts and feedback.
- Add simple save/load persistence.
- Add basic map markers for NPCs and quest objectives.
- Add placeholder art pass for Ha Giang Loop roads, villages, markets, and viewpoints.

## Phase 3: Content Expansion

- Add a second Vietnam location, such as Hanoi Old Quarter or Hoi An.
- Add new lesson categories: shopping, homestay etiquette, health, and cultural exchange.
- Expand the English Passport with location pages, badges, and review activities.

## Phase 4: Polish and Future AI Exploration

- Improve UI, animation, audio, and accessibility.
- Add spaced review for completed lessons.
- Evaluate advanced AI only after authored dialogue and progression are fun and stable.
