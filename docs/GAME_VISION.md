# Game Vision

## Working Title

**English Vietnam RPG** is a 2D top-down role-playing game where players learn practical English by traveling through culturally grounded regions of Vietnam.

## Fantasy

The player is a young traveler building an **English Passport** by helping people, completing dialogue challenges, and collecting stamps from locations across Vietnam. Each stop introduces useful English for real situations: greetings, directions, travel planning, shopping, food, safety, and storytelling.

## Player Promise

Players will feel like they are on a warm, low-pressure journey through Vietnam while steadily gaining confidence in everyday English conversations.

## MVP Setting

The first playable location is the **Ha Giang Loop**, focused on travel English, local hospitality, route planning, and mountain safety conversations.

## Learning Outcome

By the end of the MVP, players should be able to:

- Introduce themselves and ask polite questions.
- Ask for and understand simple directions.
- Order food and drinks.
- Talk about plans, weather, and safety.
- Complete short dialogue exchanges using clear English.

## Product Direction

The project should remain **data-driven**: locations, NPCs, quests, and lessons live in JSON content files so future regions can be added without rewriting core gameplay systems.
